# Tree Diagram
This is an Obsidian plugin that converts tab-indented text to a tree diagram.

## Example
Input:
~~~
```tree
/
	home
	boot
	var
		log
	usr
		local
			bin
			sbin
			lib
		bin
			cat
		sbin
	etc
```
~~~
